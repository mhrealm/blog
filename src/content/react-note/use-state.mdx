# React useState Hook 详解

## 基本用法

`useState` 是 React 中最基本的 Hook 之一，用于在函数组件中管理状态。

```jsx
import React, { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}
```

## 高级技巧

### 函数式更新

当新状态依赖于旧状态时，应该使用函数式更新：

```jsx
setCount((prevCount) => prevCount + 1);
```

### 初始化函数

当初始化值需要复杂计算时，应该使用初始化函数：

```jsx
const [count, setCount] = useState(() => {
  const initialCount = someExpensiveComputation();
  return initialCount;
});
```

## 最佳实践

1. **状态拆分**：将相关状态拆分为多个独立的 state 变量
2. **状态提升**：当多个组件需要共享状态时，将状态提升到共同的父组件
3. **使用自定义 Hook**：对于复杂的状态逻辑，考虑创建自定义 Hook

## 总结

`useState` 是 React 中管理状态的基础工具，掌握它的用法和最佳实践对于构建高效的 React 应用至关重要。
